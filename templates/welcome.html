CodeSwap is a collaborative coding solution. People can "swap" code with each other, and also chat. Right now CodeSwap supports only website creation right now but expect more soon! It's also preloaded with JQuery and Bootstrap.

You need to extend your HTML files with a .html extension otherwise Live Preview will not work.

To add Bootstrap:

<script src="/static/bootstrap/js/bootstrap.min.js"></script>
<link rel="stylesheet" href="/static/bootstrap/css/bootstrap.min.css"/>

jQuery:

<script src="/static/jquery.js"></script>

Bugs and Missing Features

  • As of the current release, you can only have .html and .css files

  • To get Javascript to work, go to Fullscreen View on your file

  • IE (Internet Explorer) and Opera do not work with this website. Using them might cause the server to crash so please upgrade to Firefox or Chrome.
